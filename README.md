# OpenGL

## Objetivo

En este repositorio está el código para la segunda parte, OpenTK/OpenGL, de la asignatura **Desarrollo de Sistemas Gráficos**, del curso 4 del **Grado en Informática de Gestión y Sistemas de Información** impartido en la **Escuela de Ingeniería de Vitoria-Gasteiz**.


## FirstSteps

Proyectos:
- FirstSteps es una aplicación de consola sencillo que crea una ventana de OpenTK. Lo usaremos para hacer pruebas mientras aprendemos a usar OpenTK/OpenGL
- OpenTKGame es el esqueleto de lo que será nuestro segundo proyecto en grupo: un juego sencillo usando OpenTK/OpenGL en 2D
- GameEngine es donde están las principales funcionalidades, algunas compartidas por los dos proyectos anteriores
